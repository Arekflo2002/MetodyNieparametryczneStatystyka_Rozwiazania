Funkcja obliczania mocy testu¶

Najoglniejsza funkcja, ktra przygotuje dane, ktre wykorzystam, aby zrobi symulacje w funkcji symulacja. Wic ustalamy liczb danych oraz rnice odchylen standardowych i idziemy dalej przekazujc parametry dalej do funkcji. Po symulacji dodajemy wynik do tablicy i idziemy dalej.

Funkcja bdzie zwraca tablice 2D(i,j), ktra zbieraa dane dla i-tej liczbie danych oraz j-tych stopni swobody oraz tablice wylosowanych stopni swobody podczas symulacjii.

def obliczanie_mocy_testu(test):
    # Tablica, ktra zbierze mi moce testw, ktre zwrci
    moce_testow = []

    for l_danych in liczby_danych:
        """
        Tworz now tablic ktra przechowuj wyniki mocu testu dla i-tej liczby danych. Lista przechowuje wyniki dla kadego j-stopnia
        swoobody w odchylenia_standardowe. Wykorzystuje List Comprahension, eby umilic kod
        """
        # Rozpoczecie symulacji 
        moce_testow.append([symulacja(l_danych,odch,test) for odch in odchylenia_std])

    # Zwracam w.w tablice
    return moce_testow

Funkcja symulacji¶

Ustalilimy ju liczb danych oraz rnice odchylen standardowych dla naszych danych. W tej funkcji bdziemy generowa dane z rozkadu normalnego dokadnie z tymi parametrami, przekazanymi jako argumenty. Nastpnie przelemy je do funkcji, ktra wybierze, ktry test jest obecnie symulowany i wynik tej funkcji bdziemy sumowa w zmiennej suma_odrzucen_H0.

Argumenty:

  • l_danych - Ile danych bdziemy losowa z rozkadu chi2
  • och - Roznica odchylen standardowych
  • test - Ktry test jest teraz symulowany

Funkcja bdzie zwraca odsetek w jakim odrzuciem hipotez zerow dla podanych parametrw

def symulacja(l_danych,odch,test):
    # W tej zmiennej bd sumowa ile razy odrzuciem hipotez zerow dla danego zestawu 
    suma_odrzucenia_H0 = 0

    # Tutaj bdzie symulacja
    for k in range(liczba_symulacji):
        # Tworz prbki danych z rozkadu normalnego o rwnym odcchyleniu standardowym
        probka1 = stats.norm.rvs(size=l_danych)
        probka2 = stats.norm.rvs(size=l_danych,scale=np.std(probka1))
        # Kolejna probka o roznym odchyleniu standardowym
        probka3 = stats.norm.rvs(size=l_danych,scale=np.std(probka1)*odch)

        # Nastpnie przekazuje wszystkie dane, aby wybrac ktry test jest symulowany i dowiedzie si jaki jest wynik testu
        suma_odrzucenia_H0 += wybor_testu(probka1,probka2,probka3,test)

    # Zwracam odsetek odrzucenia hipotezy zerowej
    return suma_odrzucenia_H0/liczba_symulacji

Funkcja wyboru testu¶

Funkcja za argumenty przyjmuje wszystkie dane, ktre s jej potrzebne do wybrania oraz przeprowadzenia danego testu. Argumenty, to:

  • probka1 - Zestaw danych wybrany we wczesniejszej funckji
  • probka2 - Zestaw danych wybrany we wczesniejszej funckji
  • test - Ktry test jest teraz symulowany

Funkcja bdzie zwraca 1 lub 0 w zalenoci od wyniku testu(1-odrzucamy test, 0-nie ma podstaw)

def wybor_testu(probka1,probka2,probka3,test):

    """
    Oglna zasada dziaania testw:

    Jeeli pvalue z danego testu jest mniejsze ni wczesniej ustalony poziom istotnotnosci to odrucamy 
    Hipoteze zerowa co znaczy ze zwracamy warto 1. W przeciwnym wypadku zwracamy 0
    
    """
    
    # Test klasyczny
    if test == 'kruskal':
        if stats.kruskal(probka1,probka2,probka3).pvalue < poziom_istotnosci: return 1
    
    # Test Manna-Whitneya
    else:
        if stats.f_oneway(probka1,probka2,probka3).pvalue < poziom_istotnosci: return 1

    # Jeeli do tej pory nie zwrcono 1 to znaczy, e aden z testw nie odrzuci H0 wic moemy zwrcic 0
    return 0
    
    # Jeeli do tej pory nie zwrcono 1 to znaczy, e aden z testw nie odrzuci H0 wic moemy zwrcic 0
    return 0

Wnioski¶

Z wykresu moemy zobaczy, e liczba danych nie ma znaczenia dla andego z testw oraz tak jak si spodziewamy przy wzrocie rnicy ilorazu odchlyen standardowych pomiedzy pierwszymi probkami a trzecia wzrasta takze moc testu. Wzrost jest bardzo podobny w obu testach(na tyle podobny, e nie da sie wyciagnac jakichkolwiek wnioskow na ten temat). Wzrost mocy testu jest te mao zauwaalny i ciezko wyciagnac jakies wnioski(wzrost tylko o 0.05), poniewa jest oczywiste, e czym wiksza rnica midzy wariancjami tym wiksza moc testu.
