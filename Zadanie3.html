Funkcja obliczania mocy testu¶

Najoglniejsza funkcja, ktra przygotuje dane, ktre wykorzystam, aby zrobi symulacje w funkcji symulacja. Wic ustalamy liczb danych oraz stopinie swobody i idziemy dalej przekazujc parametry dalej do funkcji. Po symulacji dodajemy wynik do tablicy i idziemy dalej.

Funkcja bdzie zwraca tablice 2D(i,j), ktra zbieraa dane dla i-tej liczbie danych oraz j-tych stopni swobody oraz tablice wylosowanych stopni swobody podczas symulacjii.

def obliczanie_mocy_testu():
    # Tablica, ktra zbierze mi moce testw, ktre zwrci
    moce_testow = []
    # Tablice ktra pozwoli mi zebrac stopnie swobody jakie zostaly wylosowane 
    stopnie_swobody_wylosowane = []

    # Bdzie 5 roznych losowan stopni swobody
    for i in range(5):
        """
        Tworz now tablic ktra przechowuj wyniki mocu testu dla i-tej liczby danych. Lista przechowuje wyniki dla kadego j-stopnia
        swoobody w stopnie_swobody. Wykorzystuje List Comprahension, eby umilic kod
        """
        # Losowanie stopni 
        stopien1 = random.randint(0,50)
        stopien2 = random.randint(0,50)
        stopnie_swobody_wylosowane.append([stopien1, stopien2])
        # Rozpoczecie symulacji 
        moce_testow.append([symulacja(l_danych,stopien1,stopien2) for l_danych in liczby_danych])

    # Zwracam w.w tablice
    return moce_testow,stopnie_swobody_wylosowane

Funkcja symulacji¶

Ustalilimy ju liczb danych oraz liczbe stopni swobody dla naszych danych. W tej funkcji bdziemy generowa dane z rozkadu t-studenta dokadnie z tymi parametrami, przekazanymi jako argumenty. Nastpnie przelemy je do funkcji, ktra wybierze, ktry test jest obecnie symulowany i wynik tej funkcji bdziemy sumowa w zmiennej suma_odrzucen_H0.

Argumenty:

  • l_danych - Ile danych bdziemy losowa z rozkadu chi2
  • stopien1 - Jaka jest liczba stopni swobody dla 1 probki
  • stopien1 - Jaka jest liczba stopni swobody dla 1 probki

Funkcja bdzie zwraca odsetek w jakim odrzuciem hipotez zerow dla podanych parametrw

def symulacja(l_danych,stopien1,stopien2):
    # W tej zmiennej bd sumowa ile razy odrzuciem hipotez zerow dla danego zestawu 
    suma_odrzucenia_H0 = 0

    # Tutaj bdzie symulacja
    for k in range(liczba_symulacji):
        # Tworz prbk danych z rozkadu chi2 o stopien1 stopni swobody i l_danych liczbie obserwacji
        probka = stats.chi2.rvs(df=stopien1,size=l_danych)
        # Obliczam wartosc oczekiwana
        wartosc_ocz = np.mean(probka)
        
        # Tworze 2 probke z  rozkadu chi2 o stopien2 stopni swobody i l_danych liczbie obserwacji
        probka2 = stats.chi2.rvs(df=stopien2,size=l_danych)

        # Wyrownuje wartosci oczekiwane 
        probka2 = + probka2 + (wartosc_ocz - np.mean(probka2))


        # Nastpnie przekazuje wszystkie dane, aby wybrac ktry test jest symulowany i dowiedzie si jaki jest wynik testu
        suma_odrzucenia_H0 += wybor_testu(probka, probka2)

    # Zwracam odsetek odrzucenia hipotezy zerowej
    return suma_odrzucenia_H0/liczba_symulacji

Funkcja wyboru testu¶

Funkcja za argumenty przyjmuje wszystkie dane, ktre s jej potrzebne do wybrania oraz przeprowadzenia danego testu. Argumenty, to:

  • probka1 - Zestaw danych wybrany we wczesniejszej funckji
  • probka2 - Zestaw danych wybrany we wczesniejszej funckji

Funkcja bdzie zwraca 1 lub 0 w zalenoci od wyniku testu(1-odrzucamy test, 0-nie ma podstaw)

def wybor_testu(probka1,probka2):

    """
    Oglna zasada dziaania testw:

    Jeeli pvalue z danego testu jest mniejsze ni wczesniej ustalony poziom istotnotnosci to odrucamy 
    Hipoteze zerowa co znaczy ze zwracamy warto 1. W przeciwnym wypadku zwracamy 0
    
    """
    
    # Test Kolgomorova
    if stats.kstest(probka1,probka2).pvalue < poziom_istotnosci: return 1
    
    # Jeeli do tej pory nie zwrcono 1 to znaczy, e aden z testw nie odrzuci H0 wic moemy zwrcic 0
    return 0
